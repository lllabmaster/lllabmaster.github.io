# 趋势的力量

趋势，也就是惯性，带有极大的力量。

提到惯性，我们多半会联想到物理定律，认为它只作用于物理世界；但趋势，则可以应用到任何领域。

趋势本质上就是一种惯性——当没有外力作用时，事物的发展都会按照原有的逻辑继续演进。

技术演进也是如此。每一项新技术都会经历：萌芽 → 成长 → 热潮 → 转变。前三阶段是顺应趋势，最后阶段则是外力打破。

CoT 揭开了推理模型的萌芽，行业的探索在 Deepseek-R1 之后进入成长阶段，2025 年达到热潮，而 Agent 这一外力又将推理推向更多方向（Hidden Reasoning、分层 reasoning 等）。

事物发展多半顺应趋势，但总有突发事件会改变这一轨迹。

从这个角度看，周期本质上也是一种趋势——只是它的力量更强，更不容易撼动。

# 我不看好目前的 Token 供应商

现在市场上有很多 Token 供应商，比如硅基流动、火山引擎、阿里百炼等。

我个人不太看好类似的 Token 供应模式。真正有壁垒的是闭源模型的独家供应，如 OpenAI、Anthropic、Grok 等。只有闭源才能带来足够的附加值，而通用的 Token 供应属于基础服务范畴，更适合移动、联通、电信这类低毛利企业。

同理，如果云厂商单纯卖机器，一定是亏本的。

为什么 Cloudflare 的毛利率比阿里云高很多？这一点很值得研究。

未来很长一段时间内，顶尖模型还会是闭源的，这一点几乎不可改变。道理很简单：开源的商业模式不足以产生足够的毛利来支撑持续迭代，同时开源这种「打明牌」的方式也会让闭源模型迅速借鉴其新技术。

那么为什么很多公司还要坚持走开源道路呢？打个比方：班里的前三名只需自己复习就足够了，但三到十名的同学会提议大家一起复习——因为一起复习才能接触到自己想不到的内容，而前三名本身就具备独自前行的能力。

# 模型的未来

随着技术的发展，未来的LLM将会逐渐趋同，就像移动/电信/联通的服务一样，网络/通话质量不同，但是对于用户来说，并不需要太过于关心，因为日常使用足够了，更多的是电话号码绑定了哪家的服务。

而未来的 Token 服务，只要一个域名账号即可，依赖性相距电话号码不值一提，推理成本管理将是决胜环节。
